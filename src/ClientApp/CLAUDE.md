# ClientApp

## Purpose
Cross-platform mobile and desktop client application using .NET MAUI Blazor Hybrid.

## Tech Stack
- .NET MAUI
- Blazor Hybrid (WebView)
- HttpClient for API communication

## Platforms
- iOS
- Android
- Windows
- macOS

## Key Features
- Product browsing
- Shopping cart
- User authentication
- Order history

## Note
Requires .NET MAUI workload installation.
