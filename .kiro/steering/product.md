# eShop - Reference E-Commerce Application

## Overview

eShop (AdventureWorks) is a reference .NET application demonstrating a modern e-commerce platform built with microservices architecture using .NET Aspire.

## Purpose

This is a reference implementation showcasing:
- Microservices-based architecture patterns
- .NET Aspire for cloud-native application orchestration
- Modern e-commerce workflows (catalog, basket, ordering, identity, webhooks)
- Integration patterns between distributed services

## Key Business Domains

- **Catalog**: Product browsing and management
- **Basket**: Shopping cart functionality
- **Ordering**: Order processing and management
- **Identity**: Authentication and authorization (Duende IdentityServer)
- **Webhooks**: Event notifications to external systems
- **Payment Processing**: Payment workflow handling

## Target Audience

Developers learning or implementing:
- Microservices architecture in .NET
- .NET Aspire for distributed applications
- E-commerce platform patterns
- Cloud-native application development
